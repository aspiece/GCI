Agile Brain Breaks

Five short, hands‑on activities (5–8 minutes each) to apply Agile this week. Use during transitions to reinforce habits.
Mon
Review: Story Mad Libs (Solo)
Practice writing tiny user stories about your day.
Do: Write 2 stories using the template.
  1. “As a student, I want one action so that one benefit.”
  2. Add 1 acceptance check for each (Given/When/Then or a simple pass/fail).
  3. Circle the story you could complete today.
~5 min Applies: User Stories
Alternate: One‑Sentence Stand‑up (Me)
  1. Write: Done (1 sentence), Next (1 sentence), Blocked (1 sentence).
  2. If blocked, write one person/resource that could help.
~4 minFocus: Scrum
05:00
Tue
Review: Emoji Estimation (Solo)
Size 3 small tasks from your day with S/M/L emojis.
Do: Pick 3 tasks you could do this week.
  1. Label each: 🟢 S (small), 🟡 M (medium), 🔴 L (large).
  2. Put a ⭐ next to the riskiest or most unclear one.
  3. Write 1 question you need to answer to reduce that risk.
~5 min Applies: Estimation, Risk
Alternate: Quick Slice
  1. Take one “L” task and split it into 2–3 “S” tasks.
  2. Circle the smallest slice that still gives real value.
~5 minFocus: Story Slicing
05:00
Wed
Review: Mini‑Kanban (Solo)
Build a tiny To‑Do → Doing → Done and finish one card.
Do: Draw three columns on paper or notes.
  1. Add 3 cards (one task each) to To‑Do.
  2. Set WIP=1 for “Doing”. Move exactly one card to Doing.
  3. Work 3 minutes and move it to Done. If blocked, write the blocker and 1 action to unblock.
~5 minApplies: Flow, WIP
Alternate: Bottleneck Spotter
  1. List your current steps (e.g., research → draft → review → submit).
  2. Star the slowest step and write 1 idea to speed it up.
~4 minFocus: Flow
05:00
Thu
Review: DoD 3‑Check (Solo)
Define “done” for one task and actually finish it.
Do: Pick one task and write 3 checkboxes.
  1. Example: ✅ works, ✅ reviewed, ✅ share/demo.
  2. Do the task now; tick each box as you complete it.
  3. If a box is hard, rewrite it to be clearer/smaller.
~5 minApplies: Quality (DoD)
Alternate: Bug Hunt (Solo)
  1. Find a tiny “bug” in your routine (e.g., messy folder, slow login).
  2. Write a 1‑line fix story and do the smallest fix now.
~5 minFocus: Continuous Improvement
05:00
Fri
Review: Retro Mini (Solo)
Reflect and pick one experiment for next week.
Do: Keep / Change / Start.
  1. Write one thing to Keep (worked well).
  2. Write one thing to Change (didn’t help).
  3. Write one thing to Start (new idea). Pick ONE to try next week and define “done” in 1 line.
~5 minApplies: Retrospective
Alternate: MoSCoW Me
  1. List your tasks and sort into Must / Should / Could / Won’t.
  2. Start the top Must next Monday.
~4–5 minFocus: Prioritization
05:00
How to Use
  • Pick today’s solo card. Use “Swap Activity” if you prefer the alternate.
  • Press “Start Timer” and keep it short. Finish one small outcome.
  • Write down your result (story, checkboxes, board, or experiment) to show in class.
Tip: Small wins add up. Repeat your favorite any time.
