Agile Brain Breaks

Five short, hands‑on activities (5–8 minutes each) to apply Agile this week. Use during transitions to reinforce habits.
Mon
Stand‑up Lite
One sentence each: Done / Next / Blocked.
Do: Go around quickly.
  1. Say one thing you finished.
  2. Say one thing you will do next.
  3. If blocked, name it and one person to help.
4–5 min Applies: Scrum
Alternate: High/Low + Blocker
  1. Share your top win (High) and top challenge (Low).
  2. If challenge blocks progress, write it on the board.
4 minFocus: Transparency
05:00
Tue
Tiny User Story
Write one small, testable story.
Do: Use the simple template.
  1. Write: “As a [user], I want [one action] so that [one benefit].”
  2. Add 2 acceptance criteria with Given/When/Then.
  3. Circle one story to do next.
5–6 min Applies: Stories
Alternate: INVEST Quick Fix
  1. Pick any current story.
  2. Make it Smaller or more Testable with one edit.
4–5 minFocus: Quality
06:00
Wed
Quick Estimate (S/M/L)
Vote fast to spot uncertainty.
Do: Pick one item and estimate.
  1. Everyone silently picks S / M / L.
  2. Reveal together; if split, one sentence from high/low.
  3. Agree on a size or mark “needs spike”.
4–5 minApplies: Estimation
Alternate: One Risk Tag
  1. Pick one item; add a Risk tag: Unknown / Dependency / Complex.
  2. If Risk, write one small step to reduce it.
4 minFocus: Risk First
05:00
Thu
WIP Limit Mini
Finish one thing; reduce multitasking.
Do: Set WIP=1 for “Doing” just for now.
  1. Move one in‑progress item to Done.
  2. If blocked, add a ⛔ tag and a name to help.
  3. Do not start new work until one item finishes.
4–5 minApplies: Flow
Alternate: DoD 3‑Point Check
  1. Pick one item and define 3 checks (e.g., tests, review, demo).
  2. Apply them now before moving to Done.
4–5 minFocus: Quality
05:00
Fri
Retro Lite
Pick one small improvement.
Do: Keep / Change.
  1. Each person names one thing to Keep and one to Change.
  2. Vote by hands; pick one Change.
  3. Assign one owner and define “done” in one line.
5–6 minApplies: Kaizen
Alternate: Fast MoSCoW
  1. Take 5–6 tasks; sort into Must / Should / Could / Won’t.
  2. Start the top Must next.
5 minFocus: Value
06:00
How to Use
  • Pick today’s card. Click “Swap Activity” if the alternate fits better.
  • Press “Start Timer”. Keep it short and outcome‑focused.
  • Capture one tangible change (e.g., updated order, DoD, owner on impediment).
Tip: Repeat any favorite on future weeks; consistency builds habits.
