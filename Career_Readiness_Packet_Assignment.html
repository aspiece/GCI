Career Readiness Packet

Student Assignment • Independent + Virtual

Project Goal

Create one polished 3-page packet for an internship application: a cover page, resume-style profile, and one-page cover letter.

Estimated time: 90–150 minutes total

Submit: .docx + .pdf + 150–250 word reflection

Your Completion Progress

0% complete

Page Requirements

  • Page 1: Cover page with title, name, class/date, tagline, relevant image + alt text
  • Page 2: Resume-style profile with summary, skills, education, and experience
  • Page 3: One-page cover letter to a fictional employer

Must-Show Word Skills

  • Heading styles and clear section labels
  • Bulleted list + at least one table
  • Image inserted with alt text
  • ATS-friendly formatting (simple layout, readable text)
  • Accessibility review + PDF export

Success Criteria (What Proficient Looks Like)

  • Page 1: clear visual hierarchy, clean spacing, relevant image, and purposeful alt text.
  • Page 2: easy-to-scan sections, specific action-oriented bullets, ATS-friendly formatting, and a readable table.
  • Page 3: professional tone, concise paragraphing, and complete closing/signature.

Suggested Pacing (Asynchronous)

  • Block 1 (30–45 min): Watch videos, take notes, build Page 1.
  • Block 2 (30–45 min): Build Page 2 with ATS-friendly formatting.
  • Block 3 (30–60 min): Draft Page 3, run checks, export, submit.

Example Quality Benchmarks

ATS-friendly summary example:

High school student seeking internship opportunities with strengths in communication, spreadsheet organization, and project support. Experienced through class projects and volunteer service, with consistent on-time completion and attention to detail.

Weak vs. strong bullet:

Weak: Helped with school event.
Strong: Coordinated sign-in and materials for a 120-student school event, improving check-in flow and reducing delays.
