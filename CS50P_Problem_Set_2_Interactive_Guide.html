camelCase:
Lecture tie-in: String manipulation, loops, and conditionals. See Week 2 Notes and String Methods Short.
Concept: Convert a variable name from camelCase to snake_case. Use loops to iterate through characters and conditionals to check for uppercase letters.
Worked Example:
Input: "myVariableName"
Output: "my_variable_name"
# Starter code for camelCase name = input("camelCase: ") result = "" for c in name: if c.isupper(): result += "_" + c.lower() else: result += c print(result)

Know
Review lecture concepts and string methods. These are especially useful for Problem Set 2:
Method What it does Example
str.lower() Converts all letters to lowercase "Hello".lower() → "hello"
str.strip() Removes whitespace from both ends " hi ".strip() → "hi"
str.replace(a, b) Replaces all occurrences of a with b "foo bar".replace("foo", "baz") → "baz bar"
str.split() Splits string into a list by spaces "a b c".split() → ["a", "b", "c"]
str.startswith(x) Checks if string starts with x "hello".startswith("he") → True
str.endswith(x) Checks if string ends with x "hello".endswith("lo") → True
Steps
  1. Read the input (from user or file).
  2. Process the string (use methods above).
  3. Check conditions (if/elif/else).
  4. Output the result (print or return).
