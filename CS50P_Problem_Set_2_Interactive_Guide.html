camelCase Guidance:
  1. Read the problem description: Convert a variable name from camelCase to snake_case.
  2. Loop through each character in the string.
  3. If the character is uppercase, add an underscore and its lowercase version to the result.
  4. Else, add the character as is.
  5. Print the result.
Lecture Example: See String Methods.
Tip: Use isupper() and lower() methods.
Coke Machine Guidance:
  1. Initialize amount_due to 100.
  2. Use a while loop to keep asking for coins until amount_due is 0 or less.
  3. Accept only 5, 10, or 25 cent coins.
  4. Subtract the coin value from amount_due if valid.
  5. Print the change owed at the end.
Lecture Example: See Loops.
Tip: Use while and if statements.

Starter Code: 
# Starter code for camelCase
name = input("camelCase: ")
result = ""
for c in name:
    if c.isupper():
        result += "_" + c.lower()
    else:
        result += c
print(result)
Starter Code: 
# Starter code for Coke Machine
amount_due = 100
while amount_due > 0:
    print(f"Amount Due: {amount_due}")
    coin = int(input("Insert Coin: "))
    if coin in [25, 10, 5]:
        amount_due -= coin
print("Change Owed:", abs(amount_due))
