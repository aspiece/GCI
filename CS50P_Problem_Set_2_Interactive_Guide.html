CS50P Problem Set 2 – Interactive Guide

Guidance for camelCase
View Problem Page
  1. Read the input variable name in camelCase format.
  2. Loop through each character in the string.
  3. If the character is uppercase, add an underscore and its lowercase version to the result.
  4. Else, add the character as is.
  5. Print the result in snake_case.
Guidance for Coke Machine
View Problem Page
  1. Initialize amount_due to 100.
  2. Use a while loop to keep asking for coins until amount_due is 0 or less.
  3. Accept only 5, 10, or 25 cent coins.
  4. Subtract the coin value from amount_due if valid.
  5. Print the change owed at the end.
Guidance for twttr
View Problem Page
  1. Read the input string.
  2. Loop through each character.
  3. If the character is not a vowel, add it to the result.
  4. Print the result.
Guidance for Vanity Plates
View Problem Page
  1. Read the plate input.
  2. Check if the plate is alphabetic and meets length requirements.
  3. Apply all validation rules as described in the problem.
  4. Print "Valid" or "Invalid".
Guidance for Nutrition Facts
View Problem Page
  1. Read the fruit input and convert to lowercase.
  2. Use a dictionary to map fruit names to calories.
  3. Check if the fruit is in the dictionary.
  4. Print the calories or "Item not found".
