Problem-by-Problem Guide

Problem 1 – camel.py

Concept: Character iteration

Task: Convert camelCase text into snake_case.

Lecture pattern 
for c in text:
    print(c)

c.isupper()
  1. Ask for input.
  2. Loop through each character.
  3. If uppercase, add _ then lowercase version.
  4. Otherwise add the character normally.
Official CS50 camel.py spec

Problem 2 – coke.py

Concept: while loop control

Task: Simulate a vending machine with a 50-cent price and accepted coins 25, 10, 5.

Lecture pattern 
while amount_due > 0:
    ...
  1. Start with amount_due = 50.
  2. Prompt for coin while amount due is above zero.
  3. Only subtract valid coins; ignore invalid values.
  4. Print change owed at the end.
Official CS50 coke.py spec

Problem 3 – twttr.py

Concept: Filtering characters

Task: Remove all vowels from input text.

Lecture pattern 
for c in text:
    ...
  1. Ask for input text.
  2. Loop through characters.
  3. Print or collect only non-vowel characters.
  4. Handle uppercase and lowercase vowels.
Official CS50 twttr.py spec

Problem 4 – plates.py

Concept: String validation

Task: Validate license plates against the assignment rules.

  • Starts with two letters
  • Length is 2 to 6
  • Numbers appear only at the end
  • First number cannot be 0
  • No punctuation or spaces
Lecture pattern 
s.isalpha()
s.isdigit()
Official CS50 plates.py spec

Problem 5 – nutrition.py

Concept: Dictionaries

Task: Print calories for a fruit using dictionary lookups.

Lecture pattern 
fruits = {
    "apple": 130,
    "banana": 110
}

if fruit in fruits:
    ...
  1. Create the fruit/calorie dictionary.
  2. Ask for fruit name and normalize to lowercase.
  3. If in dictionary, print calorie output format.
Official CS50 nutrition.py spec
