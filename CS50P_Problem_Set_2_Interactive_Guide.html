Python String Methods Reference:
  • isupper(): Returns True if the character is uppercase.
    c.isupper()
  • lower(): Converts a string or character to lowercase.
    name.lower()
  • isalpha(): Returns True if all characters are alphabetic.
    plate.isalpha()
  • len(): Returns the length of a string.
    len(plate)
  • in: Checks if a character or substring exists in a string.
    c in vowels
  • not in: Checks if a character or substring does not exist in a string.
    c not in vowels
  • lower(): Converts input to lowercase for case-insensitive comparison.
    fruit = input("Item: ").lower()
