Problem 1 – camel.py

Skill: Loop through letters

Your goal: Change camelCase text into snake_case.

Lecture pattern 
for c in text:
    print(c)

c.isupper()
  1. Ask the user to type text.
  2. Check one character at a time.
  3. If the character is uppercase, add _ and the lowercase letter.
  4. If not uppercase, add the character as-is.
Official CS50 camel.py spec

Problem 2 – coke.py

Skill: while loop

Your goal: Build a vending machine that starts at 50 cents and only accepts 25, 10, and 5.

Lecture pattern 
while amount_due > 0:
    ...
  1. Start with amount_due = 50.
  2. Keep asking for coins while money is still due.
  3. Only subtract valid coins; ignore anything else.
  4. When done, print the change owed.
Official CS50 coke.py spec

Problem 3 – twttr.py

Skill: Filter characters

Your goal: Remove all vowels from the text.

Lecture pattern 
for c in text:
    ...
  1. Ask the user for text.
  2. Loop through characters.
  3. Keep only characters that are not vowels.
  4. Remember both lowercase and uppercase vowels.
Official CS50 twttr.py spec

Problem 4 – plates.py

Skill: Check text rules

Your goal: Check if a license plate follows every rule.

  • Starts with two letters
  • Length is 2 to 6
  • Numbers appear only at the end
  • First number cannot be 0
  • No punctuation or spaces
Lecture pattern 
s.isalpha()
s.isdigit()
Official CS50 plates.py spec

Problem 5 – nutrition.py

Skill: Dictionary lookups

Your goal: Use a dictionary to print calories for a fruit.

Lecture pattern 
fruits = {
    "apple": 130,
    "banana": 110
}

if fruit in fruits:
    ...
  1. Make a dictionary of fruits and calories.
  2. Ask for a fruit and change it to lowercase.
  3. If the fruit exists, print its calories.
